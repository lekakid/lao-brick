# 실행화면

# 깃 메세지 컨벤션

## 커밋타입  
| 타입 | 설명 |
| :--: | :--  |
| feat | 새로운 기능 추가 |
| docs | README 수정 |
| resource | 리소스 추가 및 수정 |
| style | 코드 포맷팅, 가벼운 수정 |
| refactor | 코드 전면 수정 |
| test | 테스트, 임시 코드 |
| chore | 프로젝트 설정, 패키지 매니저 수정 |
